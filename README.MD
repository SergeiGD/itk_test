# Запуск
Для запуска проекта переименуйте .env.example в .env (при необходимости поменяв параметры).

Makefile:
- run - запуск проекта
- tests - запуск тестов
- codegen-mocks - кодген моков
- codegen-api - кодген api

После запуска проекта swagger будет доступен по адресу http://0.0.0.0:8081/ , само приложение http://0.0.0.0:8080/
